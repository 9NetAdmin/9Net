So what is HTML?

  • HTML stands for Hypertext Markup Language.
  • Is the backbone and structure of websites.
  • Text wrapped in tags. The tags change how the text function.

Now CSS

  • CSS stands for Cascading Style Sheets.
  • Changes how elements look like and where they are positioned.
  • Can be used either inside an HTML file or seperately in a CSS file.
